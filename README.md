# Robotic-project-g16

## Usage Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/LiangTian-cs/Robotic-project-g16.git
cd Robotic-project-g16
```

### 2. Open the Corresponding Task Folder

Each task has its own directory. Enter the folder of the task you want to check.

For example:

```bash
cd Task1-LightFollower/code
```

### 3. Compile and Upload the Code

1. Open the corresponding source code file in **Arduino IDE**.
2. Connect the Arduino board to the computer.
3. Select the correct board type and serial port in Arduino IDE.
4. Compile the program.
5. Upload the program to the Arduino board.

### 4. View Demonstration Videos

Demonstration videos are stored in the `videos/` folder of each task directory.

For example:

```text
Task1-LightFollower/videos/
Task2-ObstacleAvoidance/videos/
Task3-LineFollower/videos/
Task4-Comprehensive/videos/
```

If a video file is too large an external video link may be provided in the corresponding task folder.
